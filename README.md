Python project created under guidance of AP-Python Professor.
------------

A plagiarism detection tool using mutliple layers of security to define wether a set of given files has any similarities that can be seen as plagiarism.
This project uses principles like:

- list & dictionary comprehension
- Jinja
- regex
- LibCST
- visitor patterns
- transformer patterns
- AST basics
- pyspellcheck
- numerous smaller python libs
